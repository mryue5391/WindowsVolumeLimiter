# WindowsVolumeLimiter

**WindowsVolumeLimiter** 是一款基于 C++ 开发的轻量级 Windows 音量管理工具，旨在解决 Windows 系统中音量异常变化、音量被其他程序自动调整等问题。

程序允许用户自定义音量检查间隔时间（毫秒），设定目标音量后点击「锁定」，程序将在背景持续监控系统音量状态。当检测到音量偏离用户设定值时，会自动将音量恢复至指定数值，从而避免因系统设定、应用程序或其他因素导致的音量异常变化。

本程序支持 Windows 11、Windows 10 以及 Windows 8 系统，采用 C++ 编写，具有低资源占用、高执行效率等特点。

## 功能特色

* 🔊 自定义目标音量值（最大10000）
* ⏱️ 支持自定义音量检查间隔时间（ms）
* 🔒 一键锁定音量
* 🔄 自动检测并恢复异常音量
* 💻 基于 C++ 开发，执行效率高
* 🪟 支援 Windows 8 / Windows 10 / Windows 11

## 工作原理

WindowsVolumeLimiter 会按照使用者设定的时间间隔定期检测目前系统音量。

当目前音量与设定值不一致时，程序会自动呼叫 Windows 音量控制接口，将音量调整回用户指定的数值。

透过此方式，可以有效避免：

* 某些应用程序自动降低或提高音量
* 系统更新或设定变更导致音量异常
* 外部设备或驱动程序造成的音量偏移

## 使用方式

1. 启动 WindowsVolumeLimiter。
2. 设定需要固定的音量大小。
3. 设定音量检查间隔时间（单位：毫秒）。
4. 点击“锁定”按钮。
5. 程序将于背景持续监控并维持设定音量。

如需解除音量限制，只需停止程序或取消锁定即可。

## 系统需求

* **操作系统：** Windows 8 或更新版本（Windows 8 / Windows 10 / Windows 11）
* **开发语言：** C++
* **目前仅支持 Windows 平台。**

## 使用场景

* 🎧 防止耳机或音响音量异常变化
* 🎮 游戏、影音播放时保持固定音量
* 🖥️ 解决 Windows 音量自动跳动问题
* 🏢 公共计算机或展示设备音量管理
* 🔊 需要长时间维持固定音量的环境

## 注意事项

* WindowsVolumeLimiter 仅控制系统音量，不会修改音频文件或硬件配置。
* 某些具有特殊音频驱动或独占模式的程序可能无法完全受此工具控制。
* 如果系统或第三方软件具有更高优先级的音量控制机制，可能会影响工具效果。
